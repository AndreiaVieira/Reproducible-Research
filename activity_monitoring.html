Introduction

Evaluation of an individuals data collected at 5-minute intervals during the day, including the number of steps performed. Data were collected in 2012, during the months of October and November.

Data characteristic

Dataset: Activity Monitoring Data (.CSV) (https://d396qusza40orc.cloudfront.net/repdata%2Fdata%2Factivity.zip) [52K] Total: 17,568 observations

Variables:

steps: Number of steps taking in a 5-minute interval (missing values are coded as NA) date: The date on which the measurement was taken in YYYY-MM-DD format interval: Identifier for the 5-minute interval in which measurement was taken
